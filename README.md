# 1. 리액트 기초 구성
```javascript
npx create-react-app react-app
cd react-app

## 타입 스크립트 사용
yarn add typescript @types/node @types/react @types/react-dom @types/jest

## tsconfig.json 파일 추가
npx typescript --init

## 불필요한 파일 제거
yarn add scss sass
yarn add  @mui/material @emotion/react @emotion/styled

yarn add @fontsource/roboto
```
